## 🎯 Общая информация о проекте

**Название:** Format Management CRM  
**Стек технологий:**
- **Frontend:** Vue 3 + TypeScript + Vite
- **Backend:** Laravel 11 + MySQL (готов к интеграции)
- **UI:** Tailwind CSS v3.4.0
- **State Management:** Pinia
- **Routing:** Vue Router
- **i18n:** vue-i18n (русский язык по умолчанию)

**Роли пользователей:**
- 👷 Worker (Работник) - базовые права
- 👨‍💼 Manager (Менеджер) - управление заявками
- 🔐 Admin (Администратор) - полные права

## 🚀 Как запустить проект

### Frontend:
```bash
cd frontend
npm install
npm run dev
# Откроется на http://localhost:5173 (или 5174, 5175, 5176)
```

### Backend (когда будете подключать):
```bash
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
# Откроется на http://localhost:8000
```
