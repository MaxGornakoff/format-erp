# Montserrat fonts

Помести сюда свои файлы шрифта Montserrat, например:

- `Montserrat-Regular.woff2`
- `Montserrat-Medium.woff2`
- `Montserrat-SemiBold.woff2`
- `Montserrat-Bold.woff2`

Рекомендуемый формат: `woff2`.

После добавления файлов можно раскомментировать или дополнить блок `@font-face` в `src/index.css` при необходимости.
